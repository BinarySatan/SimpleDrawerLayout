##SimpleDrawerLayout

easy to use

## Layout Params

 ````
<com.github.binarysatan.SimpleDrawerLayout
        android:id="@+id/drawerLayout"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:layout_below="@id/toolBar"
        app:interspace="40dp">
````
